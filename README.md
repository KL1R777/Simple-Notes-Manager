### CLI Файловый Менеджер 

Интерфейс: CLI(В командной строк терминала)

Что умеет программа:
1. Создавать файлы(предупреждать если файл уже создан)
2. Читать полностью файл с заметками(Присутствует проверка на существования файла)
3. Добавлять новые заметки в файл(Если данный файл)
4. Удалять полностью файл
5. Бесконечный выбор действий и выход из программы по вашему желанию

Что не может программа: 
1. Удалять выбранные заметки
2. Не знаю

Как запустить программу:
В репозитории один единственный main.py
откройте его в редакторе кода и запустите
